# 설치법

## 깃 클론
```bash
git clone https://github.com/ccy00967/my-nvim.git
```

## bootstrap 실행
```bash
cd my-nvim
./bootstrap.sh
```

## 클론 파일 삭제
```bash
rm -rf my-nvim
```

## ssh 접속
```bash
sshlist # ssh 접속 리스트 출력
```

```bash
./add_ssh_host.sh # 호스트 추가
```

```bash
./delete_ssh_host.sh # 호스트 삭제
```
